3.1 GPIO

GPIO は汎用入出力端子です。 さまざまな電子部品を接続し制御するために使用できるプログラム可能な端子セットです。 各ピンを設定し、ボタンやセンサーなどからの信号(入力)を読み取り、LEDやモーターなどに信号(出力)を送信できます。 これにより、ソフトウェアで物理的な世界と対話し、制御することが可能です。

Note: WARNING - GPIO は入力電圧を3.3V まで許容しますが、これより高い電圧で使用するとプロセッサが損傷します。
